Solyos CT-e Web

Landing Page para empresa emissora de CT-e



O Solyos CT-e Web é um software 100% web que faz emissão de Conhecimento de Transporte Eletrônico para profissionais que trabalham com transporte de cargas. Foi desenvolvido o layout de uma landing page para o produto com intuito de mostrar suas funcionalidades aos clientes, explicar como funciona o software e oferecer os planos. A landing page conta com uma interface limpa, simples e intuitiva.
